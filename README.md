## SPECIFICATIONS

[1] VISUAL SYSTEM
    - Abstract geometry only
    - Palette: High-contrast monochrome, neutral greys
    - Composition: Strict grids, concentric radii, linear interconnects

[2] PARAMETERS
    - COMPLEXITY ...... Element count and layer density
    - DENSITY ......... Grid frequency and hatch modulation
    - EMPHASIS ........ Radial vs. Rectilinear balance
    - SYMMETRY ........ Axis mirroring (X / XY / Radial)
    - CONTRAST ........ Luminance threshold and background polarity
